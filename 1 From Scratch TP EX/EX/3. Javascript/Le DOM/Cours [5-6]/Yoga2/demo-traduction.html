Boutons Traduits

💡 Comment ça marche ?

1. Attribut data-i18n

<h1 data-i18n="hero.title">Prenez 15 minutes pour vous</h1>

2. Objet de traductions (JavaScript)

const translations = { fr: { "hero.title": "Prenez 15 minutes pour vous" }, sr: { "hero.title": "Odvojite 15 minuta za sebe" }, en: { "hero.title": "Take 15 Minutes for Yourself" } };

3. Sauvegarde automatique

La langue choisie est sauvegardée dans localStorage, donc elle persiste entre les sessions.

4. Changement sans rechargement

Tout le contenu change instantanément sans recharger la page !

🔧 Fichiers modifiés

  • index.html - Ajout des attributs data-i18n
  • js/language.js - Système de traduction complet
  • css/styles.css - Styles pour le bouton de langue (déjà existant)
