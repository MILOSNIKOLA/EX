# Portfolio Website

Portfolio professionnel de Milos Nikola - Développeur Web Frontend

## Description

Site portfolio moderne et responsive présentant mes compétences, mon parcours et ma vision en tant que développeur web frontend.

## Technologies utilisées

- HTML5
- CSS3 (avec variables CSS et Grid/Flexbox)
- JavaScript (vanilla)

## Structure

- `index.html` - Page principale avec toutes les sections
- `styles.css` - Styles et design responsive
- `script.js` - Interactions et navigation fluide

## Déploiement

Ce site est configuré pour GitHub Pages et accessible à l'adresse : https://milosnikola.github.io/

## Caractéristiques

- Design moderne et professionnel
- Responsive (mobile, tablette, desktop)
- Navigation fluide avec scroll smooth
- Sections bien structurées
- Mise en page claire et lisible

## Sections

1. **About Me** - Présentation personnelle
2. **My Background** - Parcours et motivation
3. **What I Do** - Compétences et focus
4. **How I Work** - Qualités et approche
5. **Purpose** - Objectifs du portfolio
6. **Personal Note** - Note personnelle
7. **My Vision** - Vision à moyen terme
8. **Contact** - Liens de contact

## Personnalisation

Pour modifier le contenu, éditez directement le fichier `index.html`. Les styles peuvent être ajustés dans `styles.css` via les variables CSS définies dans `:root`.
