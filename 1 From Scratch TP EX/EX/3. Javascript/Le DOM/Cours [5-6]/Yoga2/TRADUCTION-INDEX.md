## 🔧 Cas d'Usage Courants

### Ajouter une nouvelle traduction

1. Consultez la section "Ajouter une nouvelle traduction" dans [TRADUCTION-RECAP.md](TRADUCTION-RECAP.md)
2. Suivez les exemples de code fournis

### Ajouter une nouvelle langue

1. Lisez la section "Ajouter une nouvelle langue" dans [TRADUCTION-GUIDE.md](TRADUCTION-GUIDE.md)
2. Suivez les 3 étapes : drapeau, ordre, traductions

### Déboguer un problème

1. Consultez la section "Dépannage" dans [TRADUCTION-GUIDE.md](TRADUCTION-GUIDE.md)
2. Utilisez [test-traduction.html](test-traduction.html) pour tester

### Changer la langue par défaut

1. Voir la section "Personnalisation" dans [TRADUCTION-GUIDE.md](TRADUCTION-GUIDE.md)
2. Modifier une seule ligne dans `js/language.js`
